[兽人控小说站](https://furrynovel.com) 是一个兽人控主题的聚合小说站。这是FurryNovel.com的前端项目。

## 开始开发

```bash
pnpm install
pnpm dev
```

使用浏览器打开 http://localhost:3000 查看结果。

## 部署

只需创建PR，待合并到分支后，等待CI自动部署。
